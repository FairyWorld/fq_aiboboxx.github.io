莫急，上面这些代理协议是从应用层角度上讲的，V2Ray 强大的一点在于，它在传输层上也可以高度自定义。 同样是截止到2018年12月，V2Ray 可选的传输层配置有：用于优化网络质量的 kcp、QUIC 和 TCP Fast Open；用于伪装的 WebSocket；HTTP/2 传输；以及 TLS 加密。 不仅是 HTTP 和 SOCKS 可以在传输层配置 TLS 使其变成 HTTPS 和 SOCKS over TLS 协议，MTProto、Shadowsocks 和 VMess 也都可以通过传输层的配置包一层 TLS 在它们外面进行加密（并伪装成 TLS 流量）。HTTP 和 SOCKS 不必说，TLS 肯定是要的，VMess 裹 TLS 层也是常见的做法，但应该没人会在 Shadowsocks 外面再使用 TLS 加密。 V2Ray 客户端推荐 V2Ray
