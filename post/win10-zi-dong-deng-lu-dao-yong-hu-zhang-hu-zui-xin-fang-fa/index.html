win10自动登录到用户帐户最新方法

· 2018-12-12 · # win10 # 自动登录

如何在Win10版本2004'20H1'中自动登录到用户帐户？从Win10版本2004(也以其代码名称'20H1'已知)开始，Microsoft更改了用户自动登录功能的默认行为。现在，如果您启用了任何Windows Hello安全选项，则将无法自动登录用户帐户。为此我们给你带来简单的Win10版本2004中自动登录到用户帐户方法，轻松快速的解决问题，体验到舒适的系统玩法操作。

从Win10内部版本19033开始，如果您设置PIN或其他Windows Hello安全功能，则Win10将隐藏用户必须输入用户名和密码才能使用经典PC的选项control userpasswords2。请参见以下屏幕截图：

感谢我们的网友“ Birkuli”，我们现在知道它是操作系统的新默认行为。因此，这是使其自动登录的方法。

要在Win10版本2004中自动登录到用户帐户，

1、打开设置。

2、转到帐户>登录选项。

3、关闭右侧的Windows Hello选项。

4、按键盘上的Win+R键。运行对话框将出现在屏幕上。在“运行”框中输入以下命令：(netplwiz或control userpasswords2)。

5、找到您的用户帐户，然后在列表中选择它。您应该看到上面提到的复选框：

6、关闭用户必须输入用户名和密码才能使用此PC，然后单击“应用”按钮。

7、出现自动登录提示。

8、键入两次密码，您就完成了!

对于本地帐户和Microsoft帐户，此过程相同。

要恢复默认值，请再次运行netplwiz并打开“用户必须输入用户名和密码才能使用此PC”复选框。下次登录时，将再次要求您输入密码。

最后，您可以使用其他旧方法。但是，我不建议您使用它。我会解释原因。它在Windows NT的早期版本中可用，但今天不安全。它要求将未加密的密码存储在注册表中，第三方软件和其他用户可以读取该密码!除非您知道自己在做什么，否则不要使用此方法。

通过旧版注册表调整自动登录到用户帐户

1、打开注册表编辑器应用程序。

2、转到下面的注册表项。

HKEY_LOCAL_MACHINE \ SOFTWARE \ Microsoft \ Windows NT \ CurrentVersion \ Winlogon

3、在右侧，修改或创建一个新的字符串(REG_SZ)值“AutoAdminLogon”。设置为1。

4、创建或修改新的字符串值“ DefaultUserName”，然后键入用于自动登录的用户名。

5、在此处创建一个新的字符串值“DefaultPassword”。输入上一步中的用户帐户密码。

若要禁用通过此方法启用的自动登录，请删除DefaultPassword值并将AutoAdminLogon设置为0。

以上就是站长为大家整理的Win10版本2004中自动登录到用户帐户简述。
